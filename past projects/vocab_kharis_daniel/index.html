Webpage Vocabulary:

World Wide Web:

Also known as web or www. An internet navigation system that allows for point and click navigation of the internet at large. The web is a spider web like interconnections of millions of webpages on servers around the world. Web pages are hyperlinked to connect to each other. It is a connection of webpages accessible from anywhere in the world where there is an internet connection and a computer like device.

Web Browser:

A web access program (safari, Firefox, Chrome) that can request HTML documents from web servers and render such documents on a userâ€™s display device. A software application which allows you to view and retrieve information from the internet.

URL:

Uniform Resource Locator. The address of a web page. Consists of the domain name of a web server where a resource resides, the port address to use for communication and a directory path to access named web files or resources.

HTML:

The programing language used for writing web pages. (Hypertext Markup Language). The code that tells the browser what to do.

Web Page:

One page of a web site. A single HTML document accessible via the WWW. A web page can be linked to other pages within the web site, or anywhere on the WWW with links.

HTTP:

Also known as Hypertext Transfer Protocol, the system used by the web servers to transfer data to the internet

Home Page:

The first page of a web site, accessible through a browser. Always named index.html.

Online/Offline:

You are online when your computer is connected to the internet. A printer is online when it is connected to the computer and ready to receive information to be printed. When you are using your computer but not connected to the internet, you are offline.

Database:

A collection of information that is organized and stored so it can be easily searched.

Keywords:

The words your computer searches for to find the site you are looking for.

FTP:

File Transfer Protocol. The protocol used to transfer data (files) between computers on the internet.

SMTP:

Simple Mail Transfer Protocol. The protocol used to send emails between computers on the internet.

IP:

Internet Protocol. Every computer on the internet has a unique IP address assigned to it, such as 123.446.78.9
